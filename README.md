## How to Use

1. Enter your name on the home page
2. Click "Generate Link" to create a shareable URL
3. Copy and share the link with your special someone
4. When they visit the link, they'll see the question: "Will you be my valentine, [your name]?"
5. If they click "Yes" - confetti celebration! ðŸŽ‰
